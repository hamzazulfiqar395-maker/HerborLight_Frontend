# HerborLight_Frontend
HarborLight is a frontend community operations workspace built with React, Create React App, React Router, Bootstrap 5, and Tailwind CSS. It simulates an editorial, operations-focused dashboard for tracking funding, care relationships, field sessions, crew coordination, and workspace preferences using local storage instead of a backend.
## Features

- Landing page with updated HarborLight branding and positioning
- Mission Desk dashboard with funding and field activity summaries
- Full CRUD management for funding entries, care network records, field sessions, and crew members
- Cross-workspace search for funding, care, and field records
- Studio settings for workspace branding, theme switching, and local data reset
- Local storage persistence so changes remain after refresh
- Responsive interface using Bootstrap layout and Tailwind-powered styling

## Pages

- Landing Page
- Mission Desk
- Funding Flow
- Care Network
- Field Calendar
- Crew Board
- Signal Search
- Studio Settings
